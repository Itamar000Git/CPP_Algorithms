make
./main

* **:הרצת טסטים**:
make run_tests

תיקייה כללית הכוללת makefile וקובץ readme

## הסבר כללי על הקוד:
הקוד מחולה למספר קבצים ע"פ המחלקות, בהמשך קובץ זה נסביר על כל מחלקה בנפרד ועל אופן מימוש הפונקציות בה.
# פירוט:
* **main:**  תצוגת תכלית של הקוד
* **Edge:**  מחלקה המייצגת צלע
* **Graph**  מחלקה המייצגת גרף באמצעות רשימת שכנויות
* **praiority_Q**  מבנה נתונים המייצג תור עדיפויות
* **Queue**  מבנה נתונים המייצג תור
* **Union**  מבנה נתונים המאחסן בשיטת union
* **Algorithms**  מחלקה זו משתמשת את כל האלגוריתמים המבוקשית במטלה ( BFS , DFS , DIJKASRA , KRUSKAL , PRIM)
* **Test_prog**  מחלקת טסטים בודקת את כלל מקרי הקיצון
